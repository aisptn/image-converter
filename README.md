
# Image Converter

A web-based tool to convert and batch process images with customizable format, quality, and background color.

## Features

- **Multiple Format Support**: Convert between JPEG, PNG, and WebP
- **Quality Control**: Adjust compression quality (0.0 - 1.0) for lossy formats
- **Background Color**: Set custom background color for JPEG exports
- **Batch Processing**: Upload and convert multiple images at once
- **Preview**: View images before conversion

## Usage

1. Click **open** to select one or more images
2. Choose output **format** (left-click cycles forward, right-click cycles backward)
3. Adjust **quality** if needed (disabled for PNG)
4. Set **bg color** for JPEG backgrounds (JPEG only)
5. Click **save** to download converted images
6. Click **clear** to remove all images

## Controls

- **Open**: Load images from your device
- **Save**: Download converted images
- **Clear**: Remove all loaded images
- **Format**: Toggle between JPEG, PNG, WebP
- **Quality**: Adjust compression (left-click +0.1, right-click -0.1)
- **BG Color**: Set background color (JPEG only)
